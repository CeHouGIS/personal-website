еңЁеӨҚжқӮзҡ„иҪҜд»¶ејҖеҸ‘з”ҹе‘Ҫе‘ЁжңҹдёӯпјҢеҲ©з”Ё Docker еҜ№ OpenClaw иҝӣиЎҢе®№еҷЁеҢ–йғЁзҪІпјҢеҸҜд»Ҙжңүж•Ҳйҡ”зҰ»ејҖеҸ‘зҺҜеўғпјҢд»Һж №жң¬дёҠи§ЈеҶідҫқиө–еҶІзӘҒдёҺвҖңеңЁжҲ‘зҡ„жңәеҷЁдёҠиғҪи·‘вҖқзҡ„зҺҜеўғжұЎжҹ“й—®йўҳгҖӮжң¬ж–Үе°Ҷзі»з»ҹжҖ§ең°жўізҗҶ OpenClaw ејҖеҸ‘зҺҜеўғзҡ„жҗӯе»әжөҒзЁӢпјҢж¶өзӣ–еҹәзЎҖй•ңеғҸжһ„е»әгҖҒе®№еҷЁеҶ…зј–иҜ‘й…ҚзҪ®д»ҘеҸҠзҪ‘е…ізҪ‘з»ңи°ғиҜ•зӯүж ёеҝғзҺҜиҠӮгҖӮ

## 1. еҮҶеӨҮ Dockerfile

йҰ–е…ҲпјҢжҲ‘д»¬йңҖиҰҒе®ҡд№үдёҖдёӘеҢ…еҗ«еҹәзЎҖе·Ҙе…·й“ҫзҡ„й•ңеғҸгҖӮе»әи®®еңЁе®ҝдё»жңәеҲӣе»әдёҖдёӘдё“з”Ёзҡ„е·ҘдҪңзӣ®еҪ•пјҲдҫӢеҰӮ `~/Desktop/OpenClaw_env`пјүпјҢе№¶еңЁиҜҘзӣ®еҪ•дёӢж–°е»ә `Dockerfile` ж–Үд»¶гҖӮ

```Markdown
# 1. дҪҝз”ЁжӮЁд№ жғҜзҡ„еҹәзЎҖй•ңеғҸпјҲеҰӮ ubuntu:22.04пјү
FROM ubuntu:22.04

# 2. и®ҫзҪ®зҺҜеўғеҸҳйҮҸпјҲйҒҝе…ҚдәӨдә’ејҸе®үиЈ…ж—¶иҝӣзЁӢеҚЎжӯ»пјү
ENV DEBIAN_FRONTEND=noninteractive

# 3. и®ҫзҪ®е®№еҷЁеҶ…зҡ„й»ҳи®Өе·ҘдҪңзӣ®еҪ•
WORKDIR /app

# 4. жӣҙж–°иҪҜд»¶жәҗе№¶е®үиЈ…еҝ…иҰҒзҡ„зі»з»ҹжү©еұ•еҢ…
RUN apt-get update && apt-get install -y \
    git \
    vim \
    curl \
    build-essential \
    python3-pip \
    && rm -rf /var/lib/apt/lists/*

# 5. е®үиЈ… Python зӣёе…ідҫқиө–еә“
RUN pip install --no-cache-dir \
    numpy \
    requests \
    && echo "Extensions installed successfully."

# 6. й»ҳи®ӨеҗҜеҠЁ bash з»Ҳз«Ҝ
CMD ["/bin/bash"]
```

## 2. жһ„е»әдёҺиҝҗиЎҢе®№еҷЁ

**жһ„е»әй•ңеғҸ**

йҰ–е…ҲпјҢжҲ‘д»¬йңҖиҰҒе®ҡд№үдёҖдёӘеҢ…еҗ«еҹәзЎҖе·Ҙе…·й“ҫзҡ„й•ңеғҸгҖӮе»әи®®еңЁе®ҝдё»жңәеҲӣе»әдёҖдёӘдё“з”Ёзҡ„е·ҘдҪңзӣ®еҪ•пјҲдҫӢеҰӮ~/Desktop/OpenClaw_envпјүгҖӮз„¶еҗҺжҲ‘д»¬иҝӣе…ҘиҝҷдёӘйўқе·ҘдҪңзӣ®еҪ•е№¶жү§иЎҢжһ„е»әе‘Ҫд»ӨпјҢз”ҹжҲҗиҮӘе®ҡд№үй•ңеғҸпјҡ

```Bash
cd ~/Desktop/OpenClaw_env
docker build -t OpenClaw-env .
```

**еҲӣе»әе№¶еҗҜеҠЁе®№еҷЁ**

йҖҡиҝҮ -v еҸӮж•°е°Ҷе®ҝдё»жңәзҡ„ејҖеҸ‘зӣ®еҪ•жҢӮиҪҪеҲ°е®№еҷЁеҶ…пјҢд»Ҙе®һзҺ°е®ҝдё»жңәдёҺе®№еҷЁй—ҙзҡ„д»Јз Ғе®һж—¶еҗҢжӯҘпјҡ

```Bash
# еҗҺеҸ°еҗҜеҠЁе№¶жҢӮиҪҪзӣ®еҪ•
docker run -itd --name OpenClaw -v ~/Desktop/OpenClaw_env:/app OpenClaw-env

# иҝӣе…Ҙе®№еҷЁеҶ…йғЁзҡ„дәӨдә’ејҸз»Ҳз«Ҝ
docker start OpenClaw
docker exec -it OpenClaw /bin/bash
```

## 3. е®№еҷЁеҶ…зҺҜеўғй…ҚзҪ®
иҝӣе…Ҙе®№еҷЁеҗҺпјҢжҲ‘д»¬йңҖиҰҒжүӢеҠЁйғЁзҪІ OpenClaw зҡ„еә•еұӮзј–иҜ‘дҫқиө–д»ҘеҸҠ Node.js иҝҗиЎҢзҺҜеўғгҖӮ

**е®үиЈ…зј–иҜ‘дҫқиө–**

```Bash
apt-get update
apt-get install -y cmake build-essential libsdl2-dev libglew-dev git
```

жҲ‘д»¬е°ҶйҖүжӢ©npmе·Ҙе…·е®ҢжҲҗOpenClawзҡ„й…ҚзҪ®пјҢиҝҷжҳҜOpenClawе®ҳж–№иҫғдёәжҺЁиҚҗзҡ„еҒҡжі•гҖӮдёӢйқўжҢүз…§д»ҘдёӢжӯҘйӘӨйғЁзҪІжңҖж–°зҡ„ LTS зүҲжң¬пјҡ

```Bash
# 1. жӣҙж–°иҪҜд»¶еҢ…еҲ—иЎЁе№¶е®үиЈ… curl дёӢиҪҪе·Ҙе…·
apt-get update
apt-get install -y curl

# 2. иҺ·еҸ– Node.js 22 зүҲжң¬зҡ„е®ҳж–№е®үиЈ…и„ҡжң¬е№¶жү§иЎҢ
curl -fsSL https://deb.nodesource.com/setup_22.x | bash -

# 3. е®үиЈ… Node.jsпјҲnpm дјҡиҮӘеҠЁи·ҹйҡҸе®үиЈ…пјү
apt-get install -y nodejs
```

**е®үиЈ…дёҺеҲқе§ӢеҢ–**

еңЁиҝҷдёҖжӯҘжҲ‘д»¬е®ҢжҲҗOpenClawзҡ„й…ҚзҪ®

```Bash
npm i -g OpenClaw@latest
OpenClaw onboard
```

## 6. зҪ‘е…ій…ҚзҪ®

**и§ЈеҶізҪ‘з»ңдёҺзҪ‘е…ій“ҫжҺҘй—®йўҳ**

еҰӮжһңеңЁеҗҜеҠЁж—¶йҒҮеҲ° Health check failed: gateway closed й”ҷиҜҜпјҢиҝҷйҖҡеёёжҳҜеӣ дёәй»ҳи®Өзҡ„ loopback (localhost) з»‘е®ҡж— жі•еңЁ Docker зҡ„жЎҘжҺҘзҪ‘з»ңжЁЎејҸдёӢиў«еӨ–йғЁжӯЈеёёи®ҝй—®гҖӮ

йқўеҜ№иҝҷдёӘй—®йўҳпјҢжҲ‘д»¬йҰ–е…ҲйңҖиҰҒиҝӣе…Ҙй…ҚзҪ®зӣ®еҪ•е№¶и°ғж•ҙзҪ‘е…ізҡ„з»‘е®ҡзӯ–з•ҘгҖӮ

```Bash
cd /root/.OpenClaw/
```

е°Ҷ OpenClaw.json дёӯзҡ„ gateway еӯ—ж®өиҝӣиЎҢеҰӮдёӢдҝ®ж”№пјҡ

```JSON
// дҝ®ж”№еүҚ
"gateway": {
        "port": 18789,
        "mode": "local",
        "bind": "loopback",
....
}

// дҝ®ж”№еҗҺ
"gateway": {
        "port": 18789,
        "mode": "local",
        "bind": "lan",
....
}
```

**жүӢеҠЁеҗҜеҠЁзҪ‘е…і**

й…ҚзҪ®е®ҢжҲҗеҗҺпјҢйҖҡиҝҮд»ҘдёӢе‘Ҫд»ӨжҢҮе®ҡз«ҜеҸЈе’ҢеұҖеҹҹзҪ‘з»‘е®ҡжЁЎејҸеҗҜеҠЁзҪ‘е…іпјҡ

```Bash
OpenClaw gateway --port 18789 --bind lan
```



жіЁж„ҸпјҡеҰӮжһңзі»з»ҹжҠӣеҮә OpenClaw: access not configured зҡ„жҸҗзӨәпјҢиҜ·и®°еҪ•жҺ§еҲ¶еҸ°иҫ“еҮәзҡ„ Pairing codeпјҢе№¶еҸҠж—¶иҒ”зі» Bot з®ЎзҗҶе‘ҳе®ҢжҲҗи®ҫеӨҮжҺҲжқғгҖӮ

**еёёи§Ғз—ӣзӮ№дёҺиҝӣйҳ¶е»әи®®**

рҹ–ҘпёҸ еӣҫеҪўжҳҫзӨә (GUI) з©ҝйҖҸ
Docker е®№еҷЁй»ҳи®ӨеӨ„дәҺж— еӨҙпјҲHeadlessпјүзҠ¶жҖҒпјҢдёҚж”ҜжҢҒзӣҙжҺҘзҡ„еӣҫеҪўеҢ–иҫ“еҮәгҖӮ

Mac з”ЁжҲ·йңҖиҰҒйўқеӨ–е®үиЈ… XQuartzпјҢе№¶еңЁеҗҜеҠЁе®№еҷЁж—¶й…ҚзҪ® DISPLAY зҺҜеўғеҸҳйҮҸиҝӣиЎҢ X11 иҪ¬еҸ‘гҖӮ

е»әи®®е°Ҷе®№еҷЁзәҜзІ№дҪңдёәзј–иҜ‘е’ҢеҗҺз«ҜйҖ»иҫ‘зҡ„ејҖеҸ‘зҺҜеўғпјҢе°Ҷ GUI еүҚз«ҜиҝҗиЎҢеңЁе®ҝдё»жңәдёҠпјҢжҲ–йҮҮз”Ёиҝңз«Ҝ Web д»ӘиЎЁзӣҳиҝӣиЎҢеҸҜи§ҶеҢ–жҺ§еҲ¶гҖӮ

еёҢжңӣиҝҷзҜҮжҢҮеҚ—иғҪеё®еҠ©жӮЁеҝ«йҖҹжү«жё… OpenClaw е®№еҷЁеҢ–йғЁзҪІзҡ„йҡңзўҚпјҒеҰӮжһңжӮЁеңЁй…ҚзҪ®иҝҮзЁӢдёӯйҒҮеҲ°д»»дҪ•зҪ‘з»ңиҪ¬еҸ‘жҲ–дҫқиө–е®үиЈ…зҡ„з–‘йҡҫжқӮз—ҮпјҢж¬ўиҝҺиҝӣдёҖжӯҘдәӨжөҒгҖӮ
